## 3. 기능 구현
### 메인 페이지
  > 게시글 정렬   
  > 독서왕(랭킹)   
  > 페이지네이션   
### 게시글
  > 게시글 등록   
  > 게시글 상세 페이지 확인   
  > 댓글 등록
### 내 기록 보기
   > 로그인 시 등록 게시글 확인   
   > 게시글 수정 및 삭제
### 회원가입, 로그인, 로그아웃
  > 로그인 비밀번호 확인  
  > 회원가입 이메일, 닉네임 중복확인 
### 마이페이지
  > 등록 게시글 수로 차트 및 레벨 확인   
  > 회원정보 수정 및 탈퇴
